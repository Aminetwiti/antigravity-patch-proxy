MITM on 443 Required (Antigravity 2.2.x)
Antigravity 2.2.x REQUIRES MITM on port 443. Without it, the Language Server will continue to fail with 127.0.0.1:443 refused. Ensure the proxy is listening on 443 and the CA certificate is installed.
